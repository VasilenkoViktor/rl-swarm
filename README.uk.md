# RL Swarm (українською)

**RL Swarm** — це децентралізований обчислювальний протокол для тренування моделей з підкріпленням (reinforcement learning) у мережі **Gensyn**.

Проєкт дозволяє запускати тренування у відкритому середовищі, використовуючи незалежні ноди, які отримують винагороду за обчислювальну роботу.

## Основні компоненти

- Gensyn RL environment
- Конфігурації моделей
- Обробка задач через `swarm.pem` (ваш приватний ключ)
